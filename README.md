# Financial News Sentiment Analysis

Predicting stock price movements using news sentiment analysis and technical indicators.

## Project Overview

This project analyzes the correlation between financial news sentiment and stock price movements. It combines natural language processing for sentiment analysis with technical analysis of stock data to identify predictive relationships.

## Features

- **Sentiment Analysis**: Analyze financial news headlines using TextBlob and VADER
- **Technical Analysis**: Calculate various technical indicators (MA, RSI, MACD, Bollinger Bands)
- **Correlation Analysis**: Measure relationships between news sentiment and stock returns
- **Visualization**: Comprehensive plots and charts for data exploration
- **Modular Design**: Well-structured codebase for extensibility
